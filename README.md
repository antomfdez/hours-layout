# Hours Keyboard Layout

## Introduction
Hours is a highly optimized layout focused on modern implementations:
- Thumb key
- Adaptative swap (the biagram `bg` alone would result to `by` and the other way around to solve the `by` stretch)
- Magic key ★ (Depending on last key press acts as one key or another to solve same finger problems)
    - Magic rules:
        - r + ★ = rn
        - n + ★ = nl
        - u + ★ = u'
        - y + ★ = ys
- Repeat key ★ (Repeats the last key press unless it finds any magic rule)
